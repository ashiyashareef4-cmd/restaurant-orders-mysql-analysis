## Dataset Information

This project uses a restaurant orders dataset consisting of menu details and customer order records.

### Tables Used

menu_items
- item_name
- category
- price

order_details
- order_id
- item_name
- quantity

### Data Preparation
The dataset was cleaned using Microsoft Excel to:
- Remove duplicate records
- Standardize column names
- Ensure correct data types for analysis

### Note
The dataset structure and analysis approach were recreated based on an online tutorial for learning and portfolio purposes.
